# Snake-Game
A clone of the classic Snake Game built with HTML, CSS, and JavaScript.

A clone of the classic game **Snake Game**, written in **HTML, CSS and JavaScript**.

##Description

- Classic snake that eats apples and grows.
- Support for **Golden Apples** every 6 apples (gives 5 points and activates Super Mode).
- **Bombs** that appear randomly for challenge.
- Add **walls** every 3 points.
- Snake speed that increases as you collect points.
- Graphics with **canvas**, rounded snake and particles for effects.
- Food, Golden Apple, Game Over and bomb sounds via Web Audio API.
- Keeps **high score** in LocalStorage.
- Background snakes for beautiful visual effect.

## Instructions

1. Open the `index.html` file in a browser (Chrome/Firefox/Edge preferred).
2. Press the **PLAY** button to start the game.
3. Use the **arrows** to move the snake:
- Left: ←
- Right: →
- Up: ↑
- Down: ↓
4. Try to eat the apples without hitting the wall, your body or the bombs.
5. **Golden Apples** appear every 6 apples and give 5 points.
